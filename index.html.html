특수 기능 설명
개요
특수 기능은 PRIME, TIME, RAND, DATE 버튼으로 각각 켜고 끌 수 있습니다.
기능을 켜면 표시값이 0으로 초기화되고 해당 기능 입력 방식으로 전환됩니다.(입력형식에 맞추어 입력할것!(Error발생))
활성화된 기능 버튼은 색이 반전되어 현재 모드를 바로 확인할 수 있습니다.
같은 버튼을 다시 누르면 기능이 꺼지고 화면에 표시된 값은 유지한 채 일반 계산 모드로 돌아갑니다.
PRIME
입력한 정수가 소수인지 판별합니다.
예시: PRIME → 37 → = → 소수
TIME
시간을 HH.MM 형식으로 입력하고 + 또는 - 로 계산합니다. 결과 표시(HH:MM 형식)
예시: TIME → 04.24 + 01.36 = → 06:00
RAND
입력한 숫자를 자릿수로 보고 랜덤 숫자를 생성합니다. - 를 누르면 같은 자릿수의 음수 영역의 랜덤값을 만듭니다.
예시: RAND → 3 = → 000~999 중 하나 / RAND → 3 - = → 000~-999 중 하나
DATE
날짜를 YYYY.MM.DD 형식으로 입력한 뒤 + 또는 - 로 계산합니다. 두 번째 값은 기간 YYYY.MM.DD 또는 일수 숫자로 입력할 수 있습니다.
예시: DATE → 2023.04.12 + 5 = → 2023.04.17 / DATE → 2023.04.12 + 0003.09.12 = → 2027.01.24
